# `web-lib-demand-hello-world`

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the library in development mode.

### `npm test`

Launches the test runner.

### `npm run build`

Builds library.

## Usage

```
const webLibDemandHelloWorld = require('@karhoo/web-lib-demand-hello-world');
```
