## Installation

### Start Development
```sh
git clone git@github.com:karhoo/web-lib-demand.git
```
#### Installs all dependencies for all packages.
```shell script
yarn
```
### Available Scripts

#### `yarn test`

Launches the test runner.

#### `yarn run lint`

Runs linting.

#### `yarn run build`

Builds all packages.
