## Installation

### Start Development

```sh
git clone git@github.com:karhoo/web-lib-demand.git
npm run bootstrap
```

### Available Scripts

In the project directory, you can run:

#### `npm run bootstrap`

Installs all dependencies for all packages.

#### `npm test`

Launches the test runner.

#### `npm run lint`

Runs linting.

#### `npm run build`

Builds all packages.
